Демонстрация работы с камерой

⚠️ Для работы потребуется разрешение на доступ к камере.

GitHub Pages работает по HTTPS, поэтому доступ к медиаустройствам возможен.

Что можно делать с этими данными:

  • Создавать фото- и видеоприложения
  • Реализовать AR (дополненную реальность) с помощью библиотек типа AR.js
  • Создать видеозвонки (WebRTC)
  • Распознавание лиц или объектов (с помощью TensorFlow.js)
  • Сканирование QR-кодов

Важные ограничения:

  1. Требуется HTTPS (GitHub Pages предоставляет его автоматически)
  2. Пользователь должен дать явное разрешение
  3. На мобильных устройствах могут быть дополнительные ограничения
  4. Нельзя сохранять видео на GitHub Pages (нужен сервер для загрузки)
